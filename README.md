# wx_robot
## 依赖
python >=3.0
wechat ==3.6.0.18

## wechat下载地址
[wechat下载地址](https://github.com/crazy-bit/wx_robot/releases/download/wechat/WeChat3.6.0.18.exe)

## 功能描述
检查config.txt中配置的monitor_dir目录，当目录中的文件发生修改时，将新增内容发送到微信群room_wx_id中

## 执行
python wx_robot.py

## build.bat
build.bat生成可执行的exe
